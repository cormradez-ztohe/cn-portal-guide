# cn-portal-guide

本项目用于归档和发布多个独立 HTML 页面。所有页面均为静态资源，以仓库形式进行存储与管理，便于版本控制与持续更新。

## 项目简介

cn-portal-guide 是一个以页面归档为主要目标的仓库。仓库内包含若干独立制作的 HTML 文件，每个文件均自成一体，不依赖外部服务或特定域名。本仓库不针对任何具体网站或平台，仅作为页面资料的集合与展示。

## 目录说明

```
cn-portal-guide/
├── index.html          # 仓库入口页面（可选）
├── pages/              # 主要页面归档目录
│   ├── page-1.html
│   ├── page-2.html
│   └── ...
└── assets/             # 静态资源目录（样式、脚本、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：存放所有独立 HTML 页面文件，按需命名。
- `assets/`：存放页面引用的公共资源，如样式表、脚本文件、图片等。
- `index.html`：可选入口页面，用于导航或展示仓库概览。

## 页面归档说明

- 每个 HTML 页面均为独立完整的静态文件，可直接在浏览器中打开。
- 页面间无强制关联，可单独浏览或引用。
- 归档内容可能随时间更新或补充，不保证所有页面长期保留。
